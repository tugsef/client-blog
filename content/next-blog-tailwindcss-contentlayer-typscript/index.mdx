Web sitesi çalışmalarımı bir arada tutmak ve yazılarımdan diğer insanlarında yararlanmasını sağlamak maksadıyla oluşturulmuştur.  FocusSpark static olarak tasarlanmış bir web sitesidir. Dinamik bir web sitesi olarak hizmete açmak gibi çalışmam olacaktır. [`Markdown{:html}`](https://www.markdownguide.org/) kullanarak hazırladığım notlarımı [`.mdx{:html}`](https://mdxjs.com/) dosyası olarak contentlayer altyapısını kullanarak siteme entegre ettim.

## Getting Started

First, run the development server:

```bash
npm run dev

npx next start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx{:html}`. The page auto-updates as you edit the file.

This project uses [`next/font{:html}`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
