# React nedir?
- React JavasScript kütphanesidir.
- Facbook ekibi olarak geliştirilmiştir. Framework olarak derlenip kullanıma açılmıştır. Facbook kKurucusu Mark Zuckerberg 
- React gibi JavaScript gibi başka kütüphanelerde vardır. Bazıları vue.js, angular.js ve vb. 
- React realTime bir uygulama geliştirmek için ideal bir kütüphanedir. 
- HTML sayfasını bir internet adresine gittiğimiz zaman tamanını görürüz bu o linkte göre bileceğimiz bir sayfadır kullanıcılar için. İşte bazı uygulamalar aktif olarak yenielerinir ve bazı verilerin anlık olarak ekrana kullanıcı ile birlikte aktarılması gerekir. İşte realTme bir uygulamada burda ortaya çıkmaktadır. DOM manuple edilerek anlık olarak diğer yapılara dokunmadan tekrrar render edilmesini önleyecek şekilde bir yaoıya ihtiyaç duyulmuş ve bu sorunu çözen bir yaklaşım Framework olrak React ortaya çıkmıştır. React oluşturduğr commonet yapısı ile sayfanın sadece ilgili değişenini render ederek sayfanın etkilenmeyen diğer birleşenlerin ekrar render edilmesinin önüne geçilerek ilgili bölümün yeniden oluşturulması sağlanmıştır 
- İhtiyacınıza uygun olmayan teknolojşlerin kullanılması **over engeering(Aşırı Mühendislik)** neden olabilir. Gereksiz efor kullanılmayan özelliklere ve gereksiz maliyetede neden olabilir.
