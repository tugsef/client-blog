# RealDOM vs VirtualDOM

**RealDOM:** Sayfanın html kısmının ana gövdesidir.
**VirsualDOM** React mevcut sayfanın bir kopyasını javascript olarak elinde elinde tutar. Bu elinde oluşturulan realDOM kısmının kopyasıdır.

İşte react **VirsualDOM** ve **RealDOM** karşılaştırarak mevcut anasayfadaki değişimi kontol eder ve iki DOM arasındaki ilgili yerleri değiştirerek yansıtır.
